# Fares Akram Mohamed - Portfolio

Portfolio website for Fares Akram Mohamed, Digital Marketing Specialist, Account Manager and Team Leader.

## Project Structure

```
portfolio/
├── index.html          # Main HTML file
├── css/
│   └── style.css      # All styles
├── js/
│   └── script.js      # JavaScript functionality
├── images/            # Images folder (currently empty - using CDN)
└── README.md          # This file
```

## Features

- **Responsive Design**: Works on all devices (mobile, tablet, desktop)
- **Modern UI**: Clean, professional design with smooth animations
- **Sections**:
  - Home (Hero with stats)
  - About
  - Work Portfolio (Instagram embeds)
  - Experience Timeline
  - Skills & Tools
  - Contact Information

## Technologies Used

- HTML5
- CSS3 (Modern layouts, animations, gradients)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Work Sans)
- Font Awesome Icons

## How to Use

1. Extract the ZIP file
2. Open `index.html` in your web browser
3. Navigate through sections using the navigation menu

## External Dependencies

- Google Fonts
- Font Awesome (via CDN)
